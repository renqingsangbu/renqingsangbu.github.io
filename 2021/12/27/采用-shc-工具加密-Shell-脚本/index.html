常用参数

-e date(指定过期时间)
-m message （指定过期提示的信息）
-f script_name （指定要编译的shell路径）
-r relax security （允许该脚本在同操作系统的不同硬件平台上运行）
-v Verbose compilation （输出编译的详细情况）
-T 允许让ltrace, strace那样的程序追踪脚本运行
